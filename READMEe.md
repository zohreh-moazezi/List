
A small React + TypeScript app to view, create, edit and delete list items.

## Tech stack
- React + TypeScript (Vite)
- TailwindCSS
- react-hook-form + zod
- date-fns
- localStorage persistence

## Features
- List view with Date Created, Title, Subtitle
- Create modal (Title + Subtitle) with validation
- Edit using same modal (pre-filled)
- Delete with confirmation
- Data persists in browser localStorage

## Run locally
```bash
yarn dev
